# Hebbian CNN - 脑科学与深度学习的融合

## 项目概述

实现基于Hebbian学习规则的卷积神经网络，结合神经科学的分子机制与深度学习的特征提取。

## 核心模块

1. **hebbian_layer.py** - Hebbian卷积层实现
2. **ltp_mechanism.py** - LTP/LTD分子机制模拟
3. **train_hebbian.py** - Hebbian训练主程序
4. **train_bp.py** - 反向传播对照
5. **compare.py** - Hebbian vs BP对比分析
6. **visualize.py** - 特征可视化

## 技术方案

- Python 3.10+
- PyTorch 2.0+
- NumPy, Matplotlib
- torchvision (数据集)
